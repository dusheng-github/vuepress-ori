## 快速使用
```bash
# git配置
git config --global user.name "输入你的用户名"
git config --global user.email "输入你的邮箱"
```
### git连接远程仓库
```bash
ssh-keygen -t rsa -C "你的邮箱"
# ssh默认保存路径
cd ~/.ssh
```
进入本地`C:/Users/[用户文件夹]/.ssh` 然后将`id_rsa.pub`公钥里面的内容复制下来, 粘贴进ssh中秘钥内容中
其他的token模式, 可以授权使用
